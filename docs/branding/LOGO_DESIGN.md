# BilBiliChat Logo Design

## 设计目标
BilBiliChat 的 Logo 需要同时表达以下四个概念：

- `Bilibili`：直播、弹幕、社区识别度
- `OBS`：桌面控制台、工作流、直播工具属性
- `Danmaku`：消息流、实时互动、屏幕叠加层
- `Desktop Client`：原生应用、稳定、专业

## 图形构成建议
### 主体轮廓
- 以“对话气泡 / 显示器窗口”作为主轮廓，体现弹幕与桌面客户端
- 上方保留类似 Bilibili 电视天线的两根短线，建立品牌联想

### 内部符号
- 使用两枚霓虹圆点代表直播状态灯或弹幕节点
- 下方用一条短横线表现“消息条”或输入区
- 外围可加左右弧线，暗示 OBS 信号环或直播流

## 风格建议
- 圆角几何图形
- 深色背景搭配蓝紫粉渐变描边
- 发光但不过度科幻，保持“开源桌面工具”气质
- 适合在：
  - GitHub README
  - 应用图标延展
  - 社区头像
  - Release 封面

## 色彩建议
- 主蓝：`#00D1FF`
- 中间过渡：`#7C5CFF`
- 点缀粉：`#FB7299`
- 背景深蓝：`#081018`

## 使用规范
- 深色背景优先使用彩色渐变版
- 浅色背景可使用深色描边单色版
- 最小展示尺寸建议不小于 `32 x 32`
