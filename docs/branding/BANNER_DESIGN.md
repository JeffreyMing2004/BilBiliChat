# BilBiliChat GitHub Banner Design

## 基础规格
- 尺寸：`1280 x 640`
- 用途：GitHub 仓库首页 Banner、社交媒体头图、Release 封面
- 设计目标：体现 `专业桌面客户端`、`OBS 控制台`、`Bilibili 科技感`、`Neon UI`

## 视觉方向
- 主风格：深色科技风、低饱和背景、高亮 Neon 边缘光
- 灵感来源：`OBS Studio`、`Discord`、`Steam` 桌面客户端
- 氛围关键词：`Desktop Console`、`OBS Workflow`、`Dark Neon`、`Bilibili Tech`

## 文案布局
- 主标题：`BilBiliChat`
- 副标题：`Professional Bilibili OBS Danmaku Desktop Client`
- 能力标签：`OpenLive`、`Overlay Studio`、`Plugin System`

## 版式建议
### 左侧
- 使用一个“桌面应用窗口”透视框展示主控制台
- 顶部保留 macOS 风格 traffic lights，强化原生客户端质感
- 内部以 Sidebar + Dashboard + Overlay Panel 的结构表现主界面

### 右侧
- 放置项目标题与英文副标题
- 使用两枚能力胶囊标签，例如 `OBS Console` 与 `Neon UI`
- 可加入一行技术栈：`Tauri 2.x • Vue 3 • TypeScript • Rust`

## 色板建议
- 背景：`#07111F`、`#0B1325`、`#080B15`
- 主霓虹蓝：`#00D1FF`
- 次级霓虹紫：`#6B7CFF`
- Bilibili 粉：`#FB7299`
- 文本白：`#EAF4FF`
- 次级文本：`#96A8C6`

## 图形元素
- 半透明窗口卡片
- 发光描边与渐变边框
- 低透明度同心圆或网格，表现桌面客户端与信号流
- 简化版折线图、消息条、Overlay 卡片，暗示直播控制与弹幕流

## 输出建议
- GitHub README：优先 `SVG`
- 社交封面：导出 `PNG 2x`
- 保留安全边距：四周至少 `64px`
