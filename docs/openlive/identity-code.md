# 身份码教程

## 身份码是什么
身份码是 Bilibili OpenLive 官方用于启动直播交互会话的授权凭据。BilBiliChat 不再使用公开弹幕 WebSocket 的房间号直连方式，而是通过身份码换取官方 session。

## 在哪里填写
设置页切换到 `OpenLive` 后，会出现“OpenLive 官方身份码”输入框。

## 当前支持的能力
- 身份码输入
- 本地格式校验
- 设置持久化保存
- 身份码清空
- 应用后重连
- 当前身份码状态显示

## 状态说明
- `未填写身份码`：尚未输入，无法启动 OpenLive。
- `身份码长度过短`：本地格式校验失败。
- `身份码格式可用`：通过本地校验，可继续请求官方 `/start`。

## 切换身份码
1. 停留在 `OpenLive` Provider。
2. 覆盖输入新的身份码。
3. 点击“应用身份码并重连”。
4. 状态面板会重新进入 `starting / recovering` 等阶段。

## 安全建议
- 不要把身份码提交到 Git 仓库。
- 不要在公开截图中暴露完整身份码。
- 建议使用 BilBiliChat 设置页中的掩码显示作为日常查看方式。
