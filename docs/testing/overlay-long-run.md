# Overlay 长时间测试

## 目标
验证 `DanmuWindow` 在长时间运行时不会出现明显内存上涨、DOM 堆积、动画异常或窗口卡顿。

## 用例
- 连续运行 `2 小时`
- 普通弹幕高频输入
- 礼物与 SC 混合输入
- 切换 `obsMode / clickThrough / alwaysOnTop`
- 调整字体、阴影、透明度与动画速度

## 观察项
- CPU / 内存趋势
- 渲染帧率是否明显下降
- DOM 节点是否自动回收
- 点击穿透是否仍可正确工作
- 多显示器移动后定位是否异常

## 通过标准
- 无持续性内存泄漏迹象
- 无明显卡顿和消息堆积
- 无渲染崩溃
