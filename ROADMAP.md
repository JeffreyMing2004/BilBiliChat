# BilBiliChat Roadmap

## Vision
BilBiliChat 的目标是从“Bilibili 弹幕桌面工具”升级为“专业级 Bilibili OBS 直播控制台”。

## Phase 1 · Foundation
- [x] Tauri 2.x 多窗口桌面应用架构
- [x] Vue 3 + TypeScript + Pinia 基础工程
- [x] Public WebSocket 直播消息接入
- [x] OBS Overlay 渲染与样式引擎
- [x] GitHub Actions 自动构建与发布
- [x] Updater 基础接入

## Phase 2 · Productization
- [x] Overlay Studio 独立窗口与模板系统
- [x] Plugin SDK 基础入口
- [x] 配置导入导出基础
- [x] Crash Reporter / Crash Center
- [x] 多主题与深色科技风 UI
- [x] macOS 原生窗口行为修复

## Phase 3 · OpenLive Official
- [ ] OpenLive 生产凭证联调
- [ ] 官方事件映射完整验证
- [ ] 房间生命周期与 token 刷新闭环
- [ ] OpenLive 错误码与降级策略产品化
- [ ] Public WS / OpenLive 自动切换策略

## Phase 4 · Plugin Ecosystem
- [ ] 完整 Plugin API
- [ ] UI Hook / Overlay Hook / Message Hook
- [ ] 插件配置持久化
- [ ] 插件隔离与安全沙箱策略
- [ ] 插件市场与社区分发机制

## Phase 5 · Overlay Studio Pro
- [ ] 可视化布局吸附与参考线
- [ ] 多模板工作区
- [ ] 动画曲线与时间轴
- [ ] 多场景 Overlay Preset
- [ ] OBS Source Profile 导出

## Phase 6 · AI Plugin Layer
- [ ] AI Auto Reply Plugin
- [ ] TTS / ASR 联动
- [ ] 弹幕摘要与热点识别
- [ ] 直播间智能运营建议
- [ ] 插件式 AI Workflow Builder

## Phase 7 · Sync And Cloud
- [ ] WebDAV Sync
- [ ] GitHub Sync
- [ ] 云端模板仓库
- [ ] 插件配置同步
- [ ] Overlay Preset 跨设备同步

## Phase 8 · Multi Platform
- [ ] macOS 产品细节继续优化
- [ ] Windows 窗口与安装体验增强
- [ ] Linux 桌面兼容性验证
- [ ] ARM 设备构建与测试
- [ ] 全平台统一更新体验
